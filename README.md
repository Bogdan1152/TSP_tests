# TSP – підготовка та тестування вхідних даних

## Опис проєкту
Rепозиторій містить програмний модуль для підготовки та перевірки вхідних даних
для задачі комівояжера. Проєкт створено в межах бакалаврської роботи на тему
апробації алгоритму інтелектуальних крапель води (Intelligent Water Drops) 
для вирішення задачі комівояжера

Основна мета - модуль для підготовки та перевірки вхідних даних для задачі комівояжера,
координати,матриці та перевірка обмежень задачі. Цей модуль є основою для того, щоб 
інтегрувати алгоритм інтелектуальних крапель води та коректно провести його апробацію

## Функціональні можливості
- введення кількості міст та їх координат;
- перевірка коректності введених даних і обмежень задачі;
- заборона дублікатів координат міст;
- обчислення евклідових відстаней між містами;
- формування та виведення матриці відстаней;
- unit-тестування функцій;
- аналіз безпеки коду за допомогою SAST.

## Структура проєкту
- `tsp.py` — основний модуль з логікою обробки даних;
- `main.py` — точка входу програми;
- `test_tsp.py` — unit-тести для перевірки функціональності;
- `.gitignore` — перелік файлів, що не відслідковуються Git;
- `README.md` — опис проєкту.

## Використані технології
- Python 3.13
- pytest — для unit-тестування
- pytest-cov — для аналізу покриття коду
- Bandit — для статичного аналізу безпеки (SAST)
- matplotlib — для візуалізації
